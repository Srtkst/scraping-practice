🧱 HTMLの構造とタグ

HTML

Webページの「文章の構造」を書くための言語。見出し、段落、画像、リンクなどをタグで表します。

タグ

<h1>, <p>, <div> など、HTMLの命令のこと。タグで囲むことで意味を伝えます。

<div>

Webページ内に「箱」を作るためのタグ。中に複数の要素をまとめて入れることができます。

<span>

文章の一部だけに装飾や意味を持たせたいときに使う小さなタグ。

属性（attribute）

タグに追加情報を与える部分。例：<a href="url"> の href が属性です。

クラス（class）

HTML要素に名前をつけて管理するための属性。デザインやスクレイピングの対象指定に使います。

🔎 セレクタと要素の取得

セレクタ

HTMLの中から特定の要素を選ぶための指定方法。例：.title は class="title" の要素を意味します。

select()

複数の要素を一気に取り出すための関数。リストで返されます。

select_one()

1つだけ要素を取り出す関数。見つかった最初の要素を返します。

find()

最初に見つかった要素を返します。HTMLタグ名で指定するのが一般的。

find_all()

指定したタグ名に一致する要素をすべて取得して、リストで返します。

get()

要素の属性（hrefなど）を取得するためのメソッド。例：tag.get("href")

attrs

要素の属性を辞書形式でまとめて取得できます。例：tag.attrs["src"]

text属性

HTML要素の中身（文字）を取り出すためのプロパティ。

🐍 Pythonの基本文法

変数

データを保存するための「名前付きの箱」。例：price = 3000

リスト

複数のデータを順番に並べて管理するデータ型。例：["りんご", "バナナ", "みかん"]

辞書（dictionary）

キーと値のペアでデータを保存するデータ型。例：{"name": "りんご", "price": 100}

for文

同じ処理を繰り返すための構文。リストなどを1つずつ取り出して処理します。

if文

「もし〇〇なら」という条件をつけて処理を分けます。例：if x > 100:

print()

データを画面に表示するための関数。デバッグや確認に便利です。

int()

文字列などを整数に変換する関数。例：int("100") → 100

try...except

エラーが出そうな処理を安全に実行するための構文。エラーが起きても止まりません。

🧰 データ処理のテクニック

split()

文字列を指定した区切り文字で分割する関数。例："A,B,C".split(",") → ["A", "B", "C"]

count()

文字列の中に、特定の文字が何回出てくるかを数える関数。例："★★★☆☆".count("★") → 3

strip()

文字列の前後の空白や改行を取り除きます。

lower()

英字をすべて小文字に変換する関数。例："Hello".lower() → "hello"

isdigit()

文字列が数字だけで構成されているかを確認します。例："1234".isdigit() → True

replace()

文字列の中の一部を別の文字に置き換えます。例："¥3000".replace("¥", "") → "3000"

🌐 Webの仕組み

URL

Webページの「住所」のようなもの。例：https://example.com

HTTP

ブラウザとサーバーが通信するためのルール。スクレイピングでもこの仕組みを使います。

User-Agent

「自分がどのブラウザか」を伝えるための文字列。スクレイピングでは人間のふりをするときに使います。

robots.txt

Webサイトの管理者が「ここはスクレイピングしないで」と伝えるファイル。マナーとして確認しましょう。
