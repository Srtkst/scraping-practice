スクレイピング課題: 商品ページ

← 解説ページへ戻る

初級

商品名と価格をすべて抜き出してください。

ヒント: `.product` 要素の中に `.title` と `.price` がある

for item in soup.select(".product"):
    print(item.select_one(".title").text)

中級

2,000円以上の商品だけを抽出してください。

ヒント: `int(price.replace("¥", ""))` で数値化

price = int(item.select_one(".price").text.replace("¥", ""))
if price >= 2000:
    print(...)
