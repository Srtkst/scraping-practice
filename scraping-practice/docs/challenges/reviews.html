スクレイピング課題: レビュー一覧

← 解説ページへ戻る

初級

すべてのレビュワー名とコメントを取得してください。

ヒント: `.review` の中に `.comment` とレビュワー名があります。

for r in soup.select(".review"):
    print(r.text)

中級

★4以上のレビューのみを抽出してください。

ヒント: `.stars` の中の `★` の数を `count("★")` で数える

stars = r.select_one(".stars").text
if stars.count("★") >= 4:
    print(...)
