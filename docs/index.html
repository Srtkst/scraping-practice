スクレイピング入門

以下はローカルで実行中の練習サイトへのリンクです（Flask起動中である必要があります）。

  • 商品一覧ページ
  • ニュース記事ページ
  • レビュー一覧ページ

基本コード


import requests
from bs4 import BeautifulSoup

res = requests.get("http://localhost:5000/products")
soup = BeautifulSoup(res.text, "html.parser")

for p in soup.select(".product"):
    print(p.select_one(".title").text)
